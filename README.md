# GTA V Enhanced Edition — DMA Menu

Free external mod menu for GTA V Enhanced Edition (GTA Online).  
Runs on a second PC via PCIe DMA card. No injection. No file modification.

## Requirements

- **DMA Card** (e.g. Squirrel, LeetDMA, Captain DMA, etc.)
- **Second PC** running Windows 10/11
- **MemProcFS / VMM** drivers installed and working
- GTA V Enhanced Edition running on the target PC

---

## Setup

1. Download the latest release
2. Extract the zip to any folder
3. Make sure your DMA card is connected and MemProcFS is working
4. Launch GTA V on your gaming PC
5. Run `侠盗猎车手5.exe` on your second PC
6. Press **INSERT** to toggle the menu overlay

> The overlay renders on top of your screen. No files are touched on the game PC.

## Troubleshooting

| Problem | Fix |
|---------|-----|
| Menu doesn't open | Make sure GTA V is running and DMA is connected |
| "DMA NOT CONNECTED" | Check MemProcFS / VMM drivers, replug DMA card |
| No players on list | Wait for session to load, click Refresh |
| ESP not showing | Enable ESP in the ESP/Radar tab and check max distance |
| Config not saving | Make sure the folder is not read-only |
